# study-site-builder

## Project context
Claude Code pipeline that turns a folder of course materials (PDFs, slides, notes) into a polished multi-page study website with flashcards, practice exams, a study map, and a last-minute review page.
Runs in step-by-step interactive mode — pauses after each phase for review.
Published and actively shared with Michigan clubs. Docs and edge cases matter.

## Structure
- `skills/` — pipeline skill definitions
- `hooks/` — Claude Code hooks
- `templates/` — output templates
- `references/` — reference docs
