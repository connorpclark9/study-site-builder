# Template Placeholder Reference

Every page template uses `{{PLACEHOLDER}}` markers that the site-builder skill replaces with actual content. This document lists all placeholders across all templates.

## Universal Placeholders (all templates)

| Placeholder | Description | Example Value |
|---|---|---|
| `{{SITE_NAME}}` | Course/site display name | `DOS3704 Operations Strategy` |
| `{{THEME_CSS}}` | Relative path to theme stylesheet | `css/theme.css` |
| `{{NAV_JS}}` | Relative path to navigation script | `js/nav.js` |

## Page-Specific Placeholders

### index.html

| Placeholder | Description | Source |
|---|---|---|
| `{{COURSE_DESCRIPTION}}` | Brief course description (1-2 sentences) | User input during design phase |
| `{{QUICK_LINKS}}` | HTML for quick-link cards to selected pages | Generated by site-builder based on selected pages |

### study-map.html

| Placeholder | Description | Source |
|---|---|---|
| `{{TOC_CONTENT}}` | HTML ordered list for sidebar table of contents | Generated from `synthesis/conceptual-map.md` |
| `{{STUDY_MAP_CONTENT}}` | HTML with `<details>` sections for each topic | Generated from `synthesis/conceptual-map.md` |

### flashcards.html

| Placeholder | Description | Source |
|---|---|---|
| `{{FLASHCARD_DATA_PATH}}` | Relative path to flashcards JSON | `data/flashcards.json` |
| `{{FLASHCARDS_JS}}` | Relative path to flashcards script | `js/flashcards.js` |

### last-minute-review.html

| Placeholder | Description | Source |
|---|---|---|
| `{{REVIEW_CONTENT}}` | HTML sections for the two-column review layout | Generated from `synthesis/last-minute-review.md` |

### sample-questions.html

| Placeholder | Description | Source |
|---|---|---|
| `{{SAMPLE_QUESTIONS_CONTENT}}` | HTML question groups with spoiler answers | Generated from `study-notes/` (3-5 questions per lecture) |

### practice-exam.html

| Placeholder | Description | Source |
|---|---|---|
| `{{EXAM_TITLE}}` | Exam display title | e.g., `Practice Exam 1` |
| `{{EXAM_DESCRIPTION}}` | Brief exam description | Generated by exam-generator |
| `{{EXAM_DATA_PATH}}` | Relative path to exam JSON data | `data/exam-1.json` |
| `{{EXAM_FORMAT_CSS}}` | Relative path to exam format stylesheet | `css/exam-styles.css` |
| `{{EXAM_FORMAT_JS}}` | Relative path to exam checker script | `js/exam-checker.js` |

## Validation Rules

1. **No remaining placeholders**: After site-builder runs, search all HTML files for `{{` — if any remain, the build is incomplete.
2. **Path validity**: All path placeholders must point to files that exist in `site/`.
3. **HTML safety**: Content placeholders must produce valid HTML (no unclosed tags, no raw markdown).
